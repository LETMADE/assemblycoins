Create New Assets

Step 1 Describe Your Asset

Specify the formation details of your Asset.

Example Request
$ curl http://api.assets.assembly.com/v1/coins/prepare \
-X POST \
-d "coin_name"="KoolKoins" \
-d "issued_amount=999666" \
-d "destination_address=1C1YLvSwh2imUsGnJ8qno1XgTKZMgcTcbp" \
-d "description=Valuable and Kool!" \
-d "email=kool@assembly.com"
Arguments
name
The name of the coin
Example Response
{ "name": "KoolKoins", "issuing_private_key": "5Js13z2ix15gfQGBsSNS5jmQYUH46wNHAicfVTthvq19M8u9z7V", "minting_fee": "0.0004", "issuing_public_address": "13DZckCAHmNFJAKK2L6ZtaojRdMSpV3TJR" }
Step 2 Fund the Issuing Address with Bitcoin

Fund the Issuing Address with Bitcoin and we will create a new Colored Coin Asset on your behalf. We sent you the issuing address's private key, so you control it!



We will email you with more information about your new Asset, such as the color address.
