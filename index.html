Create your own secure, transparent, highly-liquid coin for anything, instantly.

Coins aren't just another alt-coin. Instead it's an API so anybody, anywhere can create coins for just about anything. No premining with all the benefits of using the Bitcoin blockchain.

Step 1 Issue your new coin

You'll have your minted coins in minutes. Once you do, why don't you send some to someone.

Example Request
$ curl https://api.assets.assembly.com/v1/coins \ -X POST \ -d "name=cloints"
Arguments
name
The name of the coin
Example Response
{ "name": "coints", "ticker": "Sen…", "public_address": "asdf" }
